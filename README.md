# Fake-News-Detection
A machine learning project that classifies news articles as real or fake using Natural Language Processing (NLP) techniques. The model leverages TF-IDF vectorization and Logistic Regression to analyze textual content and accurately identify misleading news articles.
# 📰 Fake News Detection using Machine Learning

## 📌 Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify news articles as **Real** or **Fake**. The model leverages **TF-IDF Vectorization** and **Logistic Regression** to analyze textual content and detect misinformation.
## 🚀 Features
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Logistic Regression classification
- Model evaluation using F1 Score and Cross-Validation
- Prediction on custom news articles
## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP

## 📊 Workflow
1. Data Collection
2. Text Preprocessing
3. TF-IDF Vectorization
4. Model Training
5. Model Evaluation
6. News Classification
## 🎯 Outcome
Developed a machine learning model capable of accurately identifying fake and real news articles using NLP techniques.
